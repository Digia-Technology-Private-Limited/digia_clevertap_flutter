## digiaclevertap

This package is a pure Dart/Flutter library for CleverTap UI actions integration.

### Usage

Add to your pubspec.yaml:

```yaml
dependencies:
	digiaclevertap:
		path: <your_local_path>
```

Import in your Dart code:

```dart
import 'package:digiaclevertap/digia_clevertap.dart';
```

See the source files for available classes and usage.
